# Week 2: Alert Priority Levels & Incident Response

## Overview
This week focuses on understanding alert priority levels, incident classification, and basic incident response procedures. Students will learn to triage alerts, document incidents, and preserve evidence.

## Learning Objectives
- Master alert priority classification systems
- Understand incident classification frameworks
- Learn basic incident response workflows
- Practice alert triage and documentation
- Develop evidence preservation skills

## Structure

### Theory
- **Alert Priority Levels**: Understanding how to prioritize security alerts
- **Incident Classification**: Categorizing and classifying security incidents
- **Basic Incident Response**: Core principles of incident response

### Practical Tasks
1. **Alert Management**: Managing and categorizing alerts in Wazuh
2. **Response Documentation**: Documenting incident responses properly
3. **Alert Triage**: Hands-on triage of security alerts
4. **Evidence Preservation**: Techniques for preserving digital evidence
5. **Capstone Project**: Comprehensive incident response scenario

## Getting Started
1. Review the theory materials in `Theory/`
2. Complete practical tasks in order: `Practical/Task_1/` through `Practical/Task_5/`
3. Screenshots and evidence files go in respective task folders
